# WhatsApp Web Hosted

A hosted wrapper for WhatsApp Web with multiple account support.

## Features
- ✅ WhatsApp Web in a clean interface
- ✅ Multiple account switcher
- ✅ Auto-refresh to keep session alive
- ✅ Fullscreen mode
- ✅ Keyboard shortcuts
- ✅ Mobile responsive

## How to Use
1. Visit: `https://[your-username].github.io/whatsapp-web-hosted/`
2. Scan QR code with your phone
3. Use sidebar to switch between accounts

## Deploy to GitHub Pages
1. Upload these files to GitHub repository
2. Go to Settings → Pages
3. Select "main" branch as source
4. Your site will be live at: `https://[username].github.io/[repository-name]/`

## Privacy Note
This is just a wrapper for https://web.whatsapp.com
All data stays with WhatsApp. No data is stored on this server.
